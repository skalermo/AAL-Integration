GraphGen module¶

GraphGen.genEdges(n, d)¶

Generate edges for graph

Parameters

  • n – Vertex count in the graph

  • d – Density - ratio of number of edges in the graph to the number of edges in a N complete graph

Returns

List of edges represented as tuples

GraphGen.genGraph(n=0, d=0.0, k=0, p=None, seed=None)¶

Generate graph random graph based on parameters Function guaranties that the chromatic number of the returned graph won’t be greater than k

Parameters

  • n – Vertex count in the graph

  • d – Graph’s density

  • k – Create k-divisible graph if k>0

  • p – Partitioning of k-divisible graph

  • seed – For test purposes

Returns

Generated graph
